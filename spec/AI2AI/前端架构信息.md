## 模块说明

### src/api/index.ts
Axios 实例配置，设置 baseURL、请求/响应拦截器，处理 JWT token 注入和统一错误处理。

### src/api/auth.ts
认证相关 API：
- `postLogin(username: string, password: string)` - 用户登录
- `postLogout()` - 用户登出
- `getMe()` - 获取当前用户信息

### src/api/employees.ts
员工管理相关 API：
- `getEmployees(params)` - 获取员工列表（支持搜索、筛选、分页）
- `getEmployeeById(id)` - 获取单个员工详情
- `createEmployee(data)` - 创建员工
- `updateEmployee(id, data)` - 更新员工信息
- `deleteEmployee(id)` - 删除员工
- `updateEmployeeStatus(id, status)` - 更新员工状态

### src/stores/auth.ts
Pinia 认证状态管理：
- `state.token: string | null` - JWT token
- `state.user: UserInfo | null` - 当前用户信息
- `actions.login(username, password)` - 登录
- `actions.logout()` - 登出
- `actions.fetchUser()` - 获取用户信息

### src/router/index.ts
Vue Router 配置：
- `/` - 首页
- `/login` - 登录页
- `/employees` - 员工管理页
- 路由守卫：未登录用户访问需要认证的页面时重定向到登录页

### src/views/Login.vue
登录页面组件：
- 用户名/密码输入框
- 记住我选项
- 登录按钮
- 表单验证和错误提示

### src/views/Home.vue
首页组件：
- 用户信息展示
- 功能菜单卡片（员工管理、组织架构、奖惩记录）
- 退出登录功能

### src/views/Employees.vue
员工列表页面组件：
- 员工列表展示（表格）
- 搜索功能（姓名、工号、部门、状态）
- 新增/编辑/删除员工
- 分页功能

### src/components/EmployeeForm.vue
员工表单组件：
- 新增/编辑员工弹窗
- 表单验证
- 支持字段：工号、姓名、性别、年龄、联系电话、邮箱、部门、岗位、职级、入职日期、状态

### src/utils/token.ts
Token 存储工具：
- 使用 localStorage 存储 token
- `getToken()` - 获取 token
- `setToken(token)` - 保存 token
- `removeToken()` - 删除 token
