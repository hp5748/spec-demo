# 需求描述 - 用户管理系统

## 项目概述
企业级用户管理系统，用于统一管理员工基本信息、组织架构及奖惩记录，支持 PC 端和移动端访问，为企业 HR 管理提供数字化支撑。

## 核心功能

### 功能1：基本信息管理
- 员工信息维护：姓名、性别、工号、年龄、联系方式、入职日期等
- 信息录入：支持单条添加和批量导入（Excel）
- 信息编辑：修改员工基本信息，保留修改历史记录
- 信息查询：支持按姓名、工号、部门等多条件搜索
- 信息导出：支持导出员工列表为 Excel 文件
- 状态管理：员工在职/离职状态变更

### 功能2：组织架构管理
- **部门管理**
  - 部门的增删改查
  - 部门层级关系支持（多级部门）
  - 部门人员统计

- **岗位管理**
  - 岗位名称定义
  - 岗位与部门的关联关系
  - 岗位人员列表

- **职级管理**
  - 职级体系定义（1-7级，每级含C-B-A）
  - 职级调整记录
  - 职级分布统计

### 功能3：奖惩记录管理
- **红事件（奖励）**
  - 奖励类型：表扬、奖金、晋升、评优等
  - 奖励记录：时间、类型、金额/级别、原因、审批人
  - 奖励统计：个人奖励汇总、部门奖励排行

- **黑事件（处罚）**
  - 处罚类型：警告、罚款、降职、辞退等
  - 处罚记录：时间、类型、金额/级别、原因、审批人
  - 处罚统计：个人处罚汇总

### 功能4：用户认证与权限
- 用户登录/登出（JWT 认证）
- 角色权限管理（RBAC）
- 操作日志记录

### 功能5：数据统计与报表
- 员工数量统计（总人数、入职/离职趋势）
- 组织架构分布图
- 奖惩数据可视化图表

## 非功能需求

### 性能要求
- 页面首次加载时间 < 2 秒
- 列表查询响应时间 < 500ms（1000 条数据内）
- 支持 100+ 并发用户
- 移动端适配流畅

### 安全要求
- 密码加密存储（bcrypt）
- API 接口鉴权
- 敏感操作审计日志
- 数据备份机制
- 防 XSS/CSRF 攻击

### 易用性要求
- 界面简洁直观，符合用户习惯
- 支持快捷键操作
- 关键操作二次确认
- 友好的错误提示
- 所有模块功能都可以跳回主页

## 用户角色

| 角色 | 描述 | 权限范围 |
|------|------|---------|
| **系统管理员** | 系统最高权限管理者 | 全部功能权限，包括角色管理、系统配置 |
| **HR 管理员** | 人力资源管理人员 | 员工信息、组织架构、奖惩记录的维护和查看 |
| **部门主管** | 各部门负责人 | 查看本部门员工信息、奖惩记录 |
| **普通用户** | 普通员工 | 查看个人信息、个人奖惩记录 |
